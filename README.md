# 📝 Blog CMS

A dynamic **Blog Content Management System (CMS)** built using **PHP and MySQL**, featuring an admin panel for managing blog posts with full CRUD functionality.

---

## 🚀 Features

* 🔐 Admin Authentication (Login System)
* 📝 Create, Read, Update, Delete (CRUD) Blog Posts
* 📂 MySQL Database Integration
* 🎨 Simple & Responsive UI
* ⚡ Lightweight and easy to run locally

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **Server:** Apache (XAMPP)

---

## ⚙️ Setup Instructions

1. Install **XAMPP**
2. Clone or download this repository
3. Move the project folder to:

   ```
   C:\xampp\htdocs\
   ```
4. Start **Apache** and **MySQL** from XAMPP Control Panel
5. Open phpMyAdmin:

   ```
   http://localhost/phpmyadmin
   ```
6. Create a database:

   ```
   blog_cms
   ```
7. Import the file:

   ```
   setup.sql
   ```
8. Run the project:

   ```
   http://localhost/blogcms
   ```
